8a) HTML ORDERED LIST

Ordered List or Numbered List :

In the ordered HTML lists, all the list items are marked with numbers by default.
It is known as numbered list also.
The ordered list starts with <ol> tag and the list items start with <li> tag.

The example for Ordered list :

  1. Krishnan
  2. Nandhini
  3. Devi
  4. Mani

There can be different types of numbered list:

  • type="1" : Numeric Number (1, 2, 3)
  • type="I" : Capital Roman Number (I II III)
  • type="i" : Small Romal Number (i ii iii)
  • type="A" : Capital Alphabet (A B C)
  • type="a" : Small Alphabet (a b c)

Examples :

type 1:

  1. Krishnan
  2. Nandhini
  3. Devi
  4. Mani

type I:

  1. Krishnan
  2. Nandhini
  3. Devi
  4. Mani

type i:

  1. Krishnan
  2. Nandhini
  3. Devi
  4. Mani

type A:

  1. Krishnan
  2. Nandhini
  3. Devi
  4. Mani

type a:

  1. Krishnan
  2. Nandhini
  3. Devi
  4. Mani

Start attribute in List :

The start attribute is used with ol tag to specify from where to start the list items.

Examples with start value 4:

type i:

  1. Krishnan
  2. Nandhini
  3. Devi
  4. Mani

type a:

  1. Krishnan
  2. Nandhini
  3. Devi
  4. Mani

Reversed attribute in Ordered List:

This is a Boolean attribute of HTML <ol> tag, and it is new in HTML5 version.
If you use the reversed attribute then it will numbered the list in descending order (7, 6, 5, 4......1).

Examples for Reverse attribute :

type i:

  1. Krishnan
  2. Nandhini
  3. Devi
  4. Mani

type a:

  1. Krishnan
  2. Nandhini
  3. Devi
  4. Mani
